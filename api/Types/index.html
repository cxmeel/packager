FlatPackage​

</>
interface FlatPackage {
Refs: {[Instance]: string}
RootRef: string
Tree: {[string]: FlatTreeNode}
}

A flat package does not have a normal tree structure. The tree is an array of nodes. Each node that has a parent has a Parent property that is a reference to the parent node's Ref property.
